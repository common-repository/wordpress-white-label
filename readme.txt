== Description ==
This is a beta, please use at own risk, White label WordPress with a plugin instead of hacking the WordPress core.

This plugin is perfect for those who use WordPress as a development environment but would like to hide the fact WordPress has been used.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload 'wordpress-white-label' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use through the 'White Label' menu in WordPress
